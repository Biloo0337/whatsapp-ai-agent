# WhatsApp AI Agent

A professional AI-powered WhatsApp Agent built in Python. This project utilizes an AI model to automate conversational replies and provide smart, contextual assistance directly on WhatsApp.

## Features
- **Automated Replies**: Seamlessly integrates with WhatsApp to auto-respond to incoming messages.
- - **AI Integration**: Powered by advanced language models for intelligent conversations.
  - - **Easy Setup**: A clean environment setup via provided scripts.
   
    - ## Installation
    - 1. **Clone the repository** (or download and extract the ZIP file):
      2. ```bash
         git clone https://github.com/Biloo0337/whatsapp-ai-agent.git
         cd whatsapp-ai-agent
         ```
         2. **Set up the virtual environment**:
         3. Run the provided initial setup script:
         4. ```bash
            .\setup_env.ps1
            ```
            Or set it up manually:
            ```bash
            python -m venv .venv
            .venv\Scripts\activate
            ```
            3. **Install dependencies**:
            4. ```bash
               pip install -r requirements.txt
               ```

               ## Usage
               Run the main file to start the agent:
               ```bash
               python main.py
               ```
               Or use the specific whatsapp execution script:
               ```bash
               python whatsapp_ai.py
               ```
